# Knowledge Base MCP Server: BAML-KB

This MCP server provides search capabilities for the "BAML-KB" knowledge base, built using LanceDB.

## Installation

1. Install dependencies:
```bash
npm install
```

2. Register with Claude Code:
```bash
claude mcp add kb-BAML-KB -- node server.js
```

## Usage

Once registered, you can use these tools in Claude Code:

### Semantic Search
For natural language queries about the content:
```bash
mcp__kb_BAML_KB__search({query: "how to implement X"})
```

You can also specify the number of results:
```bash
mcp__kb_BAML_KB__search({query: "architecture patterns", limit: 10})
```

### Statistics
Get knowledge base information:
```bash
mcp__kb_BAML_KB__stats()
```
